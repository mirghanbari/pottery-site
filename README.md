# MJ Pottery Website

React + TypeScript + Tailwind starter homepage inspired by high-art layouts but designed with clear ecommerce structure.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build production bundle:

```bash
npm run build
```
